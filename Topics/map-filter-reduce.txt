// The filter() method creates a new array by applying a conditional statement to each element.
// The filter() method creates a new array containing elements that pass a specified condition.
// If the condition returns true, the element is included in the output array; otherwise, itâ€™s excluded.
// The reduce() method reduces an array to a single value by applying a reducer function to each element.
