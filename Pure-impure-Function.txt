### **What is a Pure Function?**  

A **pure function** is a function that:  
1. **Always returns the same output for the same input.**  
2. **Does not modify anything outside its scope (no side effects).**  

✅ **Predictable**  
✅ **No Side Effects**  
✅ **Easy to Test and Debug**  

#### **Example of a Pure Function**
```javascript
function add(a, b) {
  return a + b;
}

console.log(add(2, 3)); // 5
console.log(add(2, 3)); // 5 (Always same result)
```
✔️ **Same Input, Same Output**  
✔️ **Does Not Modify External Variables**  

---

### **What is an Impure Function?**  

An **impure function** is a function that:  
1. **May return different outputs for the same input** (depends on external factors).  
2. **Modifies global variables, DOM, API calls, or logs (has side effects).**  

❌ **Unpredictable**  
❌ **Has Side Effects**  
❌ **Harder to Test and Debug**  

#### **Example of an Impure Function**
```javascript
let total = 0;

function addToTotal(value) {
  total += value; // Modifies global variable
  return total;
}

console.log(addToTotal(5)); // 5
console.log(addToTotal(5)); // 10 (Different result for same input)
```
❌ **Modifies external variable (`total`)**  
❌ **Output depends on outside state**  

---

### **Key Differences Between Pure and Impure Functions**
| Feature | Pure Function | Impure Function |
|---------|--------------|----------------|
| Output Consistency | Always same for same input | May change for same input |
| Side Effects | None | Has side effects |
| Modifies External State | No | Yes |
| Performance | More efficient | Less efficient |
| Testing | Easier | Harder |

---

### **Examples of Side Effects in Impure Functions**
- **Modifying global variables**
- **Updating the DOM**
- **Making API calls**
- **Logging to console**
- **Using `Math.random()` or `Date.now()`**  

#### **Example: API Call (Impure Function)**
```javascript
function fetchData() {
  return fetch("https://api.example.com/data") // API Call
    .then(response => response.json());
}
```
❌ **Depends on external API response (not predictable)**  

---

### **Conclusion**  
- Use **Pure Functions** whenever possible for better performance and testability.  
- Avoid **Impure Functions** unless necessary (e.g., API calls, updating UI).
